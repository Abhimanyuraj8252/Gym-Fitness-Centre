# Gym & Fitness Centre — Template 1st

Simple responsive website template for a gym/fitness centre.

## Contents

- `index.html` — landing page
- `admin.html` — admin panel template
- `member-card.html`, `privacy.html`, `terms.html`, `thank-you.html`
- `assets/css/style.css` — styles
- `assets/js/gym-core.js`, `assets/js/admin.js` — scripts

## Preview

Open `index.html` in your browser (no build step required).
